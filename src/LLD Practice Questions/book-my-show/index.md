#### Booking movie ticket application

Requirements

1. A theater has multiple screens(audi), each sreens runs a movie for a particular time, shows are run on screens
2. assuming all user are registered and logged in the application
3. once user choose a show and selects the seats for booking then we should start user booking session which means other user will not see those seats
4. after seat selection user will process for payment failure or success will happen.

5. if payment fails , after certain time or maximum retries we should mark seats available
